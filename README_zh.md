# 培训计划 (Training program)

## 几件事要知道

[Hiplot](https://hiplot.com.cn)是一个社区驱动的开源平台，用于科学数据可视化和数据挖掘。 Hiplot平台现在由openbiox社区维护，[openbiox](https://github.com/openbiox)是一个社区驱动的生物信息学创新协作小组。



## 您在这里是因为...

- 想学习生物信息学和生物统计学技能
- 想学习如何正确进行数据挖掘
- 希望为Hiplot做出贡献并在此处分享您的代码

- ...
- 想要变得更加优秀



## 培训计划包含...

有关数据可视化/数据挖掘和Hiplot开发的一系列培训课程和脚本

- 用于练习的R脚本
- 学习课程（视频和文章）
- 用于大数据分析和可视化的脚本（将由Hiplot核心团队提供并定期更新）
- 您自己想与这里的学员共享的任何脚本
